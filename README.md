## 📌 Projetos

### 📊 [Análise de Inadimplência — SQL + Python](https://github.com/mayconfelixx/analise-inadimplencia-sql-python)
Análise de padrões de inadimplência em dados financeiros usando SQL e Python.
Consultas com `JOIN`, `GROUP BY` e `CASE WHEN` em banco SQLite. Resultados visualizados com Pandas e Matplotlib em 5 análises: distribuição de status, inadimplência por cidade e renda, evolução mensal e ranking de devedores.

`Python` `SQL` `Pandas` `Matplotlib` `SQLite`
